Modern Sticky Adbar plugin allows the WordPress blog/website owner to add a advertise section.

== Description ==
Modern Sticky Adbar plugin allows the WordPress blog/website owner to add a advertise section, there is some of styles you can change them. Modern Sticky Adbar plugin has the following features.

### Features
* Enable/Disable Option
* Hide on Mobile devices.
* Hide on PC.
* On Click Cancel Option.
* Place Your Own Advertise Code.

== Installation ==

1. Click Plugins/Add New from the WordPress admin panel
1. Search for "Modern Sticky Adbar" and install
1. Configure plugin using *settings* link under plugin name or by going to Modern Adbar Options

-or-

1. Download the .zip package
1. Unzip into the subdirectory 'modern-stikcy-adbar' within your local WordPress plugins directory
1. Refresh plugin page and activate plugin
1. Configure plugin using *settings* link under plugin name or by going to Modern Adbar Options
